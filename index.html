③ 관계식 (7채널)

Ch1 — AO 침식 (COC 외층 → POE 코어 가속)
F_AO(t) = AO_RATE · t Δh_gross(t) = E_y · F_AO · 10⁴ · χ [μm] if Δh_gross ≤ h_COC₀ : t_COC = h_COC₀ − Δh_gross, t_POE = h_POE₀ else : t_COC = 0, t_POE = max(0, h_POE₀ − (Δh_gross − h_COC₀) · μ_core) 실패: t_COC ≤ 0
Ch2 — AO 거칠기 → 광산란
Δh_tot = (h_POE₀ − t_POE) + (h_COC₀ − t_COC) L_rough = k_rough · Δh_tot
Ch3 — 광학 열화 (다중 결합)
T(t) = T₀ · exp(−α · TH) [Beer-Lambert] · exp(−k_rad · Φ̇_rad · t · 100) [kGy→krad, 방사선 황변] · exp(−k_UV · Φ̇_UV · t) [UV 황변] · exp(−L_rough(t)) [거칠기 산란] · exp(−k_cont · CVCM_eff(t)) [오염 흡착] 실패: T(t) < T₀ × 0.60
Ch4 — 영률 (혼합 법칙 + 방사선 가교)
E_base = E_COC · f_COC + E_POE · f_POE E(t) = E_base · (1 + k_xl · Φ̇_rad · t · 100)
Ch5 — 정적 열응력 박리
Δα = | α_POE − α_COC | = 120 ppm/℃ σ(t) = η · E(t) · Δα · 10⁻⁶ · ΔT [MPa] 실패: σ(t) > σ_adh
Ch6 — 열사이클 피로 (Coffin-Manson + Miner)
N(t) = N_cyc · t D(t) = (N/N_ref) · (σ/σ_adh)^n 실패: D(t) ≥ 1.0
Ch7 — Tg 판정 (시간 무관)
Tg = Tg₀ + m_g · NB (b) 저온 취성: (Tg − T_MIN) > 350 → Tg가 운용 최저온보다 350℃ 이상 위 (c) 고온 연화: (T_MAX − Tg) > 0 → Tg가 운용 최고온 아래 (운용 중 고무화)
Ch8 — Outgassing (조성 의존 포화)
TML_sat = TML_POE · f_POE + TML_COC · f_COC CVCM_sat = CVCM_POE · f_POE + CVCM_COC · f_COC TML(t) = TML_sat · (1 − exp(−t/τ)) CVCM_eff = CVCM_sat · (1 − exp(−t/τ)) · (1 − γ_AO) 실패: TML(t) ≥ 1.0% OR CVCM_eff ≥ 0.1%

수명 = 어느 채널이든 임계 도달 최초 시각의 직전 step (t − dt). Ch7 (Tg) 만 시간 무관, 즉시 판정.

④ 발표용 프리셋 3가지

⚠ Danger — NB 20 / TH 400 / rPOE 50%

결과: 0년 (Tg 고온 연화)

원인: 노보넨 함량이 낮아 Tg = −15 + 4.5×20 = 75℃. 운용 최고온 150℃에서 폴리머가 Tg를 초과해 고무화 상태 → 즉시 임무 불가.

설계 메시지: NB가 너무 적으면 우주 환경 온도 범위를 폴리머의 기계적 안전 영역으로 덮을 수 없음.

⚙ COC-Heavy — NB 40 / TH 400 / rPOE 20%

결과: 약 4.3년 (열사이클 피로 박리)

원인: 외층 COC를 320μm 두껍게 → AO는 잘 막음. 하지만 COC 비율 80%로 E_base ≈ 2400 MPa로 큼. σ_eff = η · E · Δα · ΔT ≈ 4.3 MPa로 σ_adh = 5 MPa에 매우 근접 → Miner 누적 손상이 4년 만에 D=1 도달.

설계 메시지: AO 보호만 보고 COC를 늘리면 열팽창 차이 응력으로 인한 피로가 누적되어 박리.

★ Optimal — NB 40 / TH 600 / rPOE 54%

결과: 약 11.9년 (광학 열화)

원인: 총 두께 600μm로 AO 방어 (h_COC₀ ≈ 276μm), POE 비율 54%로 E_base ≈ 1390 MPa 낮춰 σ_eff = 2.5 MPa < σ_adh = 5 → 정적·피로 박리 모두 통과. AO 침식 ~12년, 광학이 비슷한 시기 한계 도달.

설계 메시지: 두께와 POE/COC 비율을 균형 조정해 7개 채널이 모두 안전 영역에서 동시에 한계에 가까워지는 점이 최적. LEO 실용성 기준 (10년+) 충족.
