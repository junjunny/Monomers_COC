POE/COC 우주 태양전지 봉지재 열화·수명 예측 모델

Team Monomers · 공학기초설계 · LEO (ISS 400km, 51.6°) · Tsim 15년

설계 목표  |  COC·POE 조성비, 총 두께, 노보넨 함량 — 세 가지 설계 변수가 LEO 봉지재 수명에 미치는 영향을 정량적으로 시각화한다. ISS와 동일한 400km 저궤도 환경 (방사선 0.5 kGy/y, UV 5256 ESH/y, 원자산소 3.28 × 10²¹ atoms/cm²/y, ΔT 300℃ 열사이클 5840회/y)을 기준으로, NASA NTRS 레퍼런스와 고분자 역학식을 결합한 8개 독립 열화 채널을 실시간 동시 연산한다.

학부 설계 검토 도구로서의 한계  |  COC–POE 계면에 대한 실험 데이터가 문헌에 부재하여 일부 상수는 추정값을 사용했으며, 실제 우주 환경은 입자 에너지 스펙트럼·궤도 경사각·태양 활동 주기에 따라 변동하지만 본 모델은 ISS 400km 대표값을 고정 상수로 사용한다. 정량적 수명 수치보다는 수명 예측의 정성적 경향성과 설계 변수 간 트레이드오프를 보여주는 데 목적이 있다.

  • E_y — AO 침식 수율 (NASA NTRS 20090034484)

  • TML / CVCM 한계값 — 1.0% / 0.1% (ASTM E595 · NASA GSFC)

⚙ 가정값 — Calibration Parameter

COC–POE 계면 실험 데이터 부재로 문헌 범위·물리적 합리성·보수적 판단에 근거해 추정한 상수.

  • k_rad, k_UV, k_rough, k_cont — 광학 열화 상수: 문헌 범위 내 가정값
  • χ = 0.2, μ_core = 1.8, η = 0.05 — 차폐·가속침식·구속완화 인자: 물리적 합리성 기반 추정
  • τ = 0.5 y, γ_AO = 0.3 — Outgassing 시정수·AO 제거율: 보수적 추정값
  • N_ref, n — Coffin-Manson 기준 사이클·응력비 지수: 폴리머 피로 문헌 부재로 인한 보수적 추정

④ 관계식 (8채널)

Ch1 — AO 침식 (COC 외층 → POE 코어 가속)
F_AO(t) = AO_RATE · t Δh_gross(t) = E_y · F_AO · 10⁴ · χ [μm] if Δh_gross ≤ h_COC₀ : t_COC = h_COC₀ − Δh_gross, t_POE = h_POE₀ else : t_COC = 0, t_POE = max(0, h_POE₀ − (Δh_gross − h_COC₀) · μ_core) 실패: t_COC ≤ 0
Ch2 — AO 거칠기 → 광산란
Δh_tot = (h_POE₀ − t_POE) + (h_COC₀ − t_COC) L_rough = k_rough · Δh_tot
Ch3 — 광학 열화 (다중 결합)
T(t) = T₀ · exp(−α · TH) [Beer-Lambert] · exp(−k_rad · Φ̇_rad · t · 100) [kGy→krad, 방사선 황변] · exp(−k_UV · Φ̇_UV · t) [UV 황변] · exp(−L_rough(t)) [거칠기 산란] · exp(−k_cont · CVCM_eff(t)) [오염 흡착] 실패: T(t) < T₀ × 0.60
Ch4 — 영률 (혼합 법칙 + 방사선 가교)
E_base = E_COC · f_COC + E_POE · f_POE E(t) = E_base · (1 + k_xl · Φ̇_rad · t · 100)
Ch5 — 정적 열응력 박리
Δα = | α_POE − α_COC | = 120 ppm/℃ σ(t) = η · E(t) · Δα · 10⁻⁶ · ΔT [MPa] 실패: σ(t) > σ_adh
Ch6 — 열사이클 피로 (Coffin-Manson + Miner)
N(t) = N_cyc · t N_ref_eff = N_ref · max(0.2, 1 − (NB − 30) · 0.02) ↑ NB↑ → 폴리머 취성↑ → 피로 한계 사이클 감소 NB=30: 기준, NB=55: N_ref × 0.5 D(t) = (N / N_ref_eff) · (σ/σ_adh)^n 실패: D(t) ≥ 1.0
Ch7 — Tg 판정 (시간 무관)
Tg = Tg₀ + m_g · NB 고온 연화: (T_MAX − Tg) > 0 → Tg가 운용 최고온 아래 (운용 중 고무화) ※ 저온 취성 검사 제거 — NB↑ 시 취성 효과는 채널 6의 N_ref 감소로 반영 ※ Tg < T_MAX 시 즉시 실패 처리는 보수적 설계 기준임. 실제로는 Tg 초과 시 탄성률 급감 및 크리프 위험이 증가하나, 우주 임무 특성상 보수적 판정을 적용함.
Ch8 — Outgassing (조성 의존 포화)
TML_sat = TML_POE · f_POE + TML_COC · f_COC CVCM_sat = CVCM_POE · f_POE + CVCM_COC · f_COC TML(t) = TML_sat · (1 − exp(−t/τ)) CVCM_eff = CVCM_sat · (1 − exp(−t/τ)) · (1 − γ_AO) 실패: TML(t) ≥ 1.0% OR CVCM_eff ≥ 0.1%

수명 = 어느 채널이든 임계 도달 최초 시각의 직전 step (t − dt). Ch7 (Tg) 만 시간 무관, 즉시 판정.
