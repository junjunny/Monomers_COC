  • MAT — 9개 그룹, 총 29개 항목

3. 시뮬레이션 코어 — simulateAt(S, t)

임의 시각 t년의 봉지재 상태를 반환하는 순수함수. 8개 결합 채널:

(1) AO 침식       : 희생층 → (소진 시) 코어층 가속 침식
(2) AO → 거칠기   : 침식량 → 표면거칠기 → 광산란 손실
(3) 광학 열화     : 두께 × 방사선 × UV × 시너지 × 거칠기 × 오염 (다중 결합)
(4) 방사선 가교   : rad → 모듈러스 ↑
(5) 열응력        : E(rad) × |Δα| × ΔT × 구속완화
(6) 피로 누적     : (cycles/ref) × (응력비)^지수    Coffin-Manson형
(7) Tg 정렬       : 운용범위 통과 / 저온 취성 / 고온 연화 3중 검사
(8) Outgassing    : 1−exp(−t/τ) → TML(두께손실) + CVCM(광학오염)   ★신규

결합 그래프

방사선 ─► 황변 ──┐
UV ───► 황변 ──┤
시너지(rad·UV)─┤
AO ─► 침식 ─► 거칠기 ─┼─► 투과율 ──► 출력 유지율
Outgassing ─► CVCM ──┘
       │      ↑ AO clean factor (LEO 한정 보정)
       └─► TML ─► 추가 두께 손실 ─┐
                                   │
AO ─► 희생층 소진 ──► 코어층 가속 ─┴─► 층 두께

방사선 ─► 가교 ─► 모듈러스 ↑ ─► 열응력 ↑ ─┬─► 정적 박리
                                           │
열사이클 횟수 ─────────────────────────────┴─► 피로 누적

Tg ─┬─► 운용범위 통과 → 강성 변화/균열
    ├─► 저온 취성 마진 부족 → 저온 파괴
    └─► 고온 연화 마진 부족 → 고온 변형

4. 수명 산정 — findLifetime(S)

dt = 0.05년 해상도로 시간 sweep, 임계값에 가장 먼저 도달한 시각 반환.

5. 실패 모드 (9종)

  1. 황변/광학 성능 저하 (이제 outgassing 오염 포함)
  2. 태양전지 출력 감소
  3. AO 희생층 소진
  4. 핵심 기능층 손상 (AO + outgassing 두께 손실)
  5. 정적 박리 응력 초과
  6. 열사이클 피로 파괴
  7. 운용 온도 내 Tg 통과
  8. 저온 취성 위험
  9. 고온 연화 위험

구조: ✅ 즉시 제작 가능

모든 매직넘버는 ENV.LEO / MAT 두 객체에 노출되어 있어, 데이터 출처 정리되면 값 교체만으로 바로 반영됩니다.

내용 점검 시 주의할 4가지

  1. 응력 단위 관점: E·Δα·ΔT는 이미 [MPa]. thermalStressScale은 단위 환산이 아닌 구속 완화 / 응력 완화 / 표면 응력 농도를 묶은 경험적 보정 인자. 보통 0.001~0.1.
  2. 차폐 인자 위치 의존: shieldingFactor는 외부 표면이면 ≈ 1.0, 셀 사이 내부면이면 작아짐. 봉지재 위치 확정 후 값 결정.
  3. Tg 회귀식 placeholder: 현재 Tg = -15 + 4.8·NB는 NB=20에서 81°C, NB=55에서 249°C로 LEO 운용범위에 항상 걸리는 비현실적 값. 실측치 1순위 교체.
  4. kSynergy: 방사선·UV 시너지 데이터가 없으면 0으로 두는 게 안전. 추정값 넣지 말 것.

LEO 한정의 의미

실제 우주용 태양전지 위성은 대부분 LEO에 존재 (운용 위성 수 80%+). Starlink, OneWeb, ISS, 허블, 큐브셋, 지구관측 위성 모두 LEO. AO는 LEO 고유 환경이며, 폴리머 봉지재가 가장 가혹하게 시험되는 곳이라 이 프로젝트의 1차 타겟으로 적절. GEO/심우주는 별도 최적화 영역.
