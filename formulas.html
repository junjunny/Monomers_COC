4. 핵심 식 (8개 결합 채널)

1AO 침식 (POE 희생 + COC 가속)

FAO(t) = Φ̇AO · t
Δhgross(t) = Ey · FAO(t) · 10⁴ · χ    [μm]

tPOE(t) = max(0, hPOE,0 − Δhgross(t))
if Δhgross > hPOE,0:
  tCOC(t) = max(0, hCOC,0 − (Δhgross − hPOE,0) · μcore)

초기 두께: hPOE,0 = TH·rPOE/100, hCOC,0 = TH·rCOC/100

2AO → 표면 거칠기 → 광산란

Δhtot(t) = (hPOE,0 − tPOE) + (hCOC,0 − tCOC)
Lrough(t) = krough · Δhtot(t)

3광학 열화 (다중 결합 multiplicative)

T(t) = T₀ · exp(−α·TH) · exp(−krad·Φ̇rad·t)
       · exp(−kUV·Φ̇UV·t)
       · exp(−ksyn·Φ̇rad·Φ̇UV·t²)
       · exp(−Lrough(t))
       · exp(−kcont · CVCMeff(t))

각 항이 독립 채널이라 곱으로 결합. Beer-Lambert 일반화 형태.

4방사선 가교 → 영률 증가 (혼합 법칙)

Ebase = ECOC · (rCOC/100) + EPOE · (rPOE/100)
E(t) = Ebase · (1 + kxl · Φ̇rad · t)

5열응력 (CTE 불일치)

Δα = |αPOE − αCOC| = 120 ppm/℃
σfull(t) = E(t) · Δα · ΔT
σ(t) = σfull(t) · η    [MPa]

σfull은 완전 구속 가정 응력. η는 다층 구속 완화·응력 완화·표면 응력 농도를 묶은 경험 인자.

6열사이클 피로 누적 (Coffin-Manson + Miner)

N(t) = Ncyc · t
D(t) = (N(t) / Nref) · (σ(t) / σadh)n

실패 조건: D(t) ≥ 1.0. 정적 박리: σ(t) > σadh 즉시 fail.

7Tg 정렬 (3중 검사)

Tg = Tg0 + mg · NB    [℃]

(a) 운용범위 통과: Tmin < Tg < Tmax → 사이클 중 상변화 실패
(b) 저온 취성: (Tg − Tmin) > 200℃ → 저온 균열
(c) 고온 연화: (Tmax − Tg) < 20℃ → 고온 변형

8Outgassing (조성 의존 saturation × transient)

TMLsat = TMLPOE · fPOE + TMLCOC · fCOC
CVCMsat = CVCMPOE · fPOE + CVCMCOC · fCOC

TML(t) = TMLsat · (1 − exp(−t/τ))
CVCMeff(t) = CVCMsat · (1 − exp(−t/τ)) · (1 − γAO)

Δhoutgas(t) = TH · TML(t)/100    (양 층에 비례 배분)

fPOE = rPOE/100, fCOC = rCOC/100. AO가 외부 응축물을 제거 → 유효 CVCM 감소 (LEO 한정 보정).

5. 수명 산정 — 직렬 신뢰성 (최단)

Lfinal = mint ∈ [0, Tsim] { t | 어느 한 채널이라도 실패 임계 도달 }

탐색: dt = 0.05년 해상도로 sweep, 첫 실패 시각 반환.
어떤 채널도 임계 미도달 시 Lfinal = Tsim(=15년) 반환.

9가지 실패 모드를 매 시각 t에 동시 평가:

  1. 광투과율 T(t) < 80%
  2. 출력 유지율 P/P₀ < 85%
  3. POE 희생층 소진 (tPOE ≤ 0)
  4. COC 기능층 임계 (tCOC < 40μm)
  5. 정적 박리 (σ > σadh)
  6. 피로 누적 (D ≥ 1.0)
  7. Tg 운용범위 통과
  8. 저온 취성 마진 초과
  9. 고온 연화 마진 미달

7. 검증 — 파라미터 영역별 dominant 실패 모드

구성예측 수명한계 메커니즘해석
기본: NB=35, TH=400, POE=50%~12.6년AO 침식POE(200μm) 8.7년 → COC 가속 침식 4년
얇은 막: TH=200, POE=50%~5.8년AO 침식POE 100μm는 4.4년에 소진
두꺼운 막: TH=800, POE=50%~9.0년광학 (두께 페널티)Beer-Lambert α·TH가 초기 투과율 낮춤
COC-heavy: POE=20%~3.1년피로 누적Ebase 높음 → σ 증가 → D 가속
POE-heavy: POE=80%~14년AO + 광학두꺼운 희생층이 14년 보호, 그러나 outgassing 증가
저 NB=200년 (즉시)Tg 운용범위 통과Tg=75℃, 운용 −150~+150 범위 안 통과
설계 의의: 슬라이더 3개가 각각 다른 실패 모드를 dominant로 만들어 트레이드오프를 실증.
· NB: 너무 낮으면 Tg 운용범위 통과 → 즉시 실패. 충분히 높여야 함.
· TH: 얇으면 AO에 빠르게 침식, 두꺼우면 광학 손실. 중간 두께가 최적.
· POE 비율: 너무 낮으면 피로 파괴 (rigid), 너무 높으면 outgassing/광학 오염. 50% 부근이 균형.
